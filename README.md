# SOCIETYPRO 🏢
**Smart Society Management System**

SOCIETYPRO is a professional React-based web application designed to streamline communication and management within residential societies. It features a modern Glassmorphism UI and real-time data handling using Firebase.

---

## ✨ Key Features

### 🔐 Multi-Role Access
* **Manager Dashboard:** Exclusive access to manage residents, post notices, and update society rules.
* **Resident Portal:** View society updates, check maintenance bills, and participate in community chats.
* **Security:** Auto-logout feature for residents if their access is revoked by the manager.

### 💬 Real-Time Communication
* **Community Chat:** Instant messaging for all society members powered by Firebase Realtime Database.
* **Message Controls:** Edit your messages or delete them (Managers can delete any message for everyone).
* **Smart Notifications:** Real-time updates for new notices and billing changes.

### 🚨 Safety & Finance
* **Emergency SOS:** One-tap SOS button that broadcasts the user's live location via Google Maps.
* **Emergency Contacts:** Direct links to Police (100) and Ambulance (102).
* **Maintenance Tracking:** Digital maintenance balance display with integrated UPI payment links.

---

## 🛠️ Tech Stack
* **Frontend:** React.js (Functional Components, Hooks)
* **Backend:** Firebase Realtime Database
* **Styling:** CSS3 (Glassmorphism, Responsive Grid)

---

## 🚀 How to Run Locally
1. Clone the project.
2. Run `npm install` to install dependencies.
3. Add your `firebaseConfig` in `App.js`.
4. Run `npm start`.
